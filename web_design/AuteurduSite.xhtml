La création du site

Ce site est le résultat d’un travail pratique de deux étudiants de l’Université de Montréal pour le cours de design web IFT1005 du professeur Guy Lapalme. Ce site web est construit dans le respect des critères ergonomiques vus dans le cours. Toutes les pages sont écrites en français avec des vocabulaires compréhensibles. Les barres de navigation sont visibles sur toutes les pages qui sont toutes homogènes pour les 7 pages du travail. Toutes les pages sont validées avec le validateur (Cependant, le validateur trouve non nécessaire l'ajout du type de JavaScript dans la balise script. On l'a dû garder selon les modèles vus en classe) avec une mise en page bien lisible tout en gardant un design original et esthétique.
Vous pouvez accéder à nos pages de design et d'analyse à partir de la barre de navigation ou en appuyant directement sur le mot.
